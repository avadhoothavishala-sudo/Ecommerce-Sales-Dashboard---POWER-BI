## Project Overview
This project focuses on building an interactive Power BI dashboard to analyze e-commerce sales data and provide clear insights into business performance.

The dashboard helps track key metrics such as Total Sales, Profit, Orders, Average Order Value (AOV), and Profit Margin, enabling data-driven decision-making.

## Problem Statement
The business had sales data but lacked a centralized view to monitor performance, profitability, and customer trends. Decisions were made without clear visibility into high-performing regions and products.

## Data Used
- Orders data (Order ID, Date, Customer, City, State)
- Sales details (Amount, Profit, Quantity, Category, Sub-category, Payment Mode)

## Key Features
- KPI tracking for Sales, Profit, Orders, AOV, and Profit Margin  
- Category-wise and state-wise sales analysis  
- Payment mode distribution  
- Monthly sales trend analysis  
- Interactive slicers for dynamic filtering  

## Tools & Technologies
- Power BI  
- DAX  
- Excel / CSV  

## Insights
- Identified top-performing states contributing the highest revenue.  
- Highlighted loss-making sub-categories despite high sales volume.  
-Observed a growing preference for digital payment methods over cash-on-delivery.

## Impact
The dashboard enables stakeholders to:
- Focus on high-value regions and products  
- Improve pricing and profitability strategies  
- Make faster, data-driven business decisions
